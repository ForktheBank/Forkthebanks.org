# Fork the Banks — Site

## Estrutura de arquivos

```
forkthebanks/
├── index.html       ← Página principal (Homepage)
├── sobre.html       ← Sobre / Missão / Time
├── insight.html     ← Blog / Research / Insights
├── css/
│   └── style.css    ← Design system completo
├── js/
│   └── main.js      ← Animações, scroll, interações
└── README.md
```

## Design

- **Fontes**: Syne (display/títulos) + DM Mono (labels, tags) + DM Sans (body)
- **Paleta**: Preto profundo `#080A09` + Verde cripto `#00E87A` + Off-white `#F0F4F1`
- **Estética**: Fundo de investimentos institucional (Galaxy/Bitwise) com identidade Fork the Banks

## Páginas criadas

| Página | Conteúdo |
|--------|----------|
| `index.html` | Hero com stats animados, manifesto, soluções (Fork Capital / Fork Finance / Recursos), seção sobre, insights recentes, CTA, footer |
| `sobre.html` | Missão, valores, time com 4 cards, CTA "Trabalhe conosco" |
| `insight.html` | Destaque em grid, lista completa de artigos com filtros por categoria, newsletter |

## Como usar

Basta abrir `index.html` em qualquer navegador moderno.  
Não requer servidor — funciona como site estático.

## Para expandir

- Adicionar `fork-capital.html` e `fork-finance.html` para as páginas de produtos
- Adicionar `recursos.html` com glossário e calendário econômico
- Integrar formulário de contato via Formspree ou similar
- Conectar newsletter via Mailchimp/Resend
